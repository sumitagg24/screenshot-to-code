# Screenshot-to-Code | Resume Bullet Points

## 🎯 Project Title
**Screenshot-to-Code: AI-Powered Design-to-Code Generator**

---

## 📝 One-Liner Description
Full-stack AI web application that converts design screenshots into production-ready code using multiple LLMs (OpenAI, Anthropic, Google Gemini).

---

## 💡 What It Does (3 Sentence Summary)
Developed a full-stack application that leverages advanced AI models to automatically convert visual designs, mockups, and Figma designs into clean, functional code. The application supports multiple code frameworks (React, Vue, Bootstrap, HTML+Tailwind) and provides real-time WebSocket-based code generation with iterative refinement capabilities. Successfully integrated with OpenAI GPT-4, Anthropic Claude, and Google Gemini APIs with a production-ready codebase featuring 110+ unit tests.
