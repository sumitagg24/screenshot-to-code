================================================================================
                         📊 PROJECT OVERVIEW
================================================================================

PROJECT TITLE:
Screenshot-to-Code: AI-Powered Design-to-Code Generator

WHAT IT DOES:
Converts design screenshots, mockups, and Figma designs into production-ready 
code using multiple AI models (OpenAI, Anthropic, Google Gemini).

KEY FEATURES:
✓ Real-time WebSocket code generation
✓ Multi-framework support (React, Vue, Bootstrap, HTML+CSS, etc.)
✓ 5+ AI model integrations
✓ Multi-variant code generation
✓ Iterative code refinement
✓ Video-to-code conversion (experimental)
✓ Image generation with DALL-E/Flux
